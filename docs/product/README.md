# Product Docs

These files are the living product contract for `dnd_kit`.

- `overview.md`
- `package-architecture.md`
- `api-principles.md`
- `release-roadmap.md`

`SPEC.md` is retained as the seed input and historical reference. New work
should update these smaller product docs, story packets, validation proof, and
decision records instead of extending the monolithic spec.
