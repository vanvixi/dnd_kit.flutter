# Basic Drag Drop Example

Minimal documentation target for the core Flutter drag/drop API shape.

The main package now exposes:

- `DndScope` for controller ownership;
- `DndDraggable` for drag sources;
- `DndDroppable` for drop targets;
- `DndDragOverlay` for independent active-drag visuals.

This folder is reserved for a small runnable app that demonstrates those APIs
without sortable or Kanban-specific behavior. Until that app is added, use the
package README and `examples/kanban_board` for working usage patterns.
