# Examples

Example apps are added story by story as behavior becomes real.

Planned examples include:

- basic drag and drop;
- drag overlay;
- sortable list;
- sortable grid;
- Kanban board (`examples/kanban_board`);
- experimental multi-container sortable (`examples/multi_container_sortable`);
- custom collision;
- custom sensor;
- auto-scroll;
- keyboard accessibility.
